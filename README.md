Face Detection Project

Built a face detection system using Python and OpenCV for identifying human faces in images and live video streams.

Utilized pre-trained computer vision models (Haar Cascade) to achieve accurate face localization.

Performed image preprocessing, frame extraction, and real-time detection with efficient processing pipelines.

Improved detection accuracy under varying lighting and viewing conditions.

Technologies Used
Python
OpenCV
NumPy
Computer Vision
Haar Cascade Image Processing
